Function looks for cycles in graph \textquotesingle{}g\textquotesingle{} ~\newline
 First we look for a good starting vertex, one with only outgoing edges ~\newline
 If none are found, take any vertex with outgoing edges ~\newline
 If we still cant find a vertex, the graph is disconnected and non-\/cyclic

Apply Breadth First Search from the starting vertex, marking the vertexes that were already visited. If we pass by an already marked vertex, the ~\newline
