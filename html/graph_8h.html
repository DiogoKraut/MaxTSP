◆ hasCycle()

int hasCycle ( graph_t *  g)

Function looks for cycles in graph 'g'
First we look for a good starting vertex, one with only outgoing edges
If none are found, take any vertex with outgoing edges
If we still cant find a vertex, the graph is disconnected and non-cyclic

Apply Breadth First Search from the starting vertex, marking the vertexes that were already visited. If we pass by an already marked vertex, the
graph has a cycle

Parameters
gPointer to graph
Returns
1 If 'g' is cyclic
0 If 'g' not non-cyclic
